Contenido responsivo

El diseño web responsive o adaptativo es una técnica de diseño web que busca la correcta visualización de una misma página en distintos dispositivos. Desde computadoras de escritorio a tablets y celulares.

El diseño responsive permite reducir el tiempo de desarrollo, evita los contenidos duplicados, y aumenta la viralidad de los contenidos ya que permite compartirlos de una forma mucho más rápida y natural.

Bloques de contenido

La inegración de contenido se realiza utilizando bloques de contenido, existe un bloque de contenido para cada necesidad, desde bloques básicos como texto hasta bloques más complejos como actividades interactivas.

Recuros multimedia

Recursos multimedia para e-learning. La Información multimedia se refiere a un conjunto de diferentes tipos básicos de contenido, como texto, audio, imágenes, animación y vídeos. Cada uno de ellos requiere una representación de datos diferente para almacenamiento y transmisión.

Actividades interactivas

Cuando le añadimos la palabra “interactiva” a actividades las orientamos al mundo online, a la formación e-Learning.

A través de los bloques de contenido podemos acceder a la interfaz para la confuguración del contenido del tipo de actividad interactiva, en este panel seleccionamos el tipo de actividad, el tipo de recursos que utilizará y si será evaluable o no.

Dark mode

En el aspecto más práctico significa ahorro de batería. Las interfaces oscuras requieren menos energía.

El Modo Oscuro también es ideal para ambientes poco iluminados.
